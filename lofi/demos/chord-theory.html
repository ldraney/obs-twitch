Chord Reference

Major

Root + M3 + P5
Bright, happy, resolved. Less common in lofi.

Minor

Root + m3 + P5
Sad, dark, melancholic. The lofi foundation.

Major 7th

Root + M3 + P5 + M7
Dreamy, nostalgic. THE lofi sound.

Minor 7th

Root + m3 + P5 + m7
Mellow, smooth, introspective.

Dominant 7th

Root + M3 + P5 + m7
Bluesy, tense, wants to resolve.

Major 9th

Root + M3 + P5 + M7 + M9
Ethereal, floating, expansive.

Minor 9th

Root + m3 + P5 + m7 + M9
Deep, contemplative, rich.
